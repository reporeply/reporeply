About RepoReply

RepoReply is an automated repository maintenance tool designed to reduce noise and improve project hygiene. By tracking issue and pull request activity, RepoReply identifies stale items, notifies contributors with gentle reminders, and optionally closes inactive threads after a configurable grace period.

Built as a secure GitHub App, RepoReply uses scoped permissions, short-lived tokens, and webhook-based events to operate safely across multiple repositories while respecting pinned items, security-related labels, and maintainer-defined exclusions.
